# Password Generator

Generate Password and copy to clipboard

## Requirements

No extra requirements

## How to use

```bash
python scripts/pypass.py -l <size> -<flags>
```

## Flags

**-a:** All Characters
**-l:** Include lowercase
**-u:** Include uppercase
**-n:** Include numbers
**-p:** Include punctuation
**-b:** Include spaces
**-v:** Show password in terminal
